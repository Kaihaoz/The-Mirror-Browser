# The Mirror Browser
面向算法透明度的用户行为追踪与隐私可视化交互产品

## 项目介绍
这是我独立完成的交互产品项目，基于研究性设计（RtD）方法构建。
我设计并实现了一套完整的可交互原型，能够实时追踪用户网页行为（点击、悬停、滚动深度），并通过三层透明度体系可视化算法逻辑，解决用户因算法“黑盒”产生的隐私焦虑问题。

项目平衡隐私合规与用户体验，探索了算法透明化在国内互联网产品中的落地路径。

## 核心工作
- 独立完成产品设计、交互逻辑规划与前端开发实现
- 基于 HTML5/JavaScript 构建实时用户行为追踪系统（点击、悬停时长、滚动深度、停留时长）
- 设计三层算法透明度体系：微解释、实时画像侧边栏、数据仪表盘
- 使用 Chart.js 实现雷达图、热力图等多维度可视化
- 招募 29 名目标用户完成测试，基于反馈优化交互逻辑
- 总结出可复用的算法透明化设计框架

## 技术栈
HTML5、CSS3、JavaScript、Chart.js、用户测试、交互设计、数据可视化

## 项目价值
为算法透明化与隐私体验优化提供可落地的设计方案，可广泛应用于隐私合规产品、数据平台、内容产品与文创交互工具。
